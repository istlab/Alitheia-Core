First System User

The system user management implies that on system bootstrapping enough groups are available with the relevant privileges.

Example

For a freshly deployed and running system, the following groups should be available:

Group Name
new users
alitheia

For those initial groups the privileges will be the following:

Group Privilege Value On Service
new usersuser_id4svc://sqooss.security
alitheiaactionwritesvc://sqooss.security
