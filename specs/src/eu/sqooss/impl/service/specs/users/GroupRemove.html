Removing Groups from the System

In order to allow user access, and to revoke their privileges easily, the system should allow to remove user groups.

The system can remove groups from its groups list. The users should be kept, they only loose privileges.

Example

Given the system only having the initial group list (see this spec), and an extra group named "humongous", containing the following users:

User
padams
sebas

Then the initial user list is the following:

User Name Groups
alitheiaalitheia, new users
padamshumongous, new users
sebashumongous, new users

When removing the group, the users aren't removed, so the final user list will be:

User Name Groups
alitheiaalitheia, new users
padamsnew users
sebasnew users

And the final group list will be again the following:

Group Name
new users
alitheia
